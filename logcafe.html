log cafe / サービスサイト(架空)

URL

https://yahoo.co.jp

概要

緑に囲まれたコワーキングスペースとカフェが併設された店舗の架空サイト。職業訓練校の卒業制作。

ターゲット

リモートワークの会社員・フリーランサー・都会で働くことに疲れている人

デザインについて

メインカラーに若草色を、モチーフに淡い水彩ブラシを使うことで、緑に囲まれた心地の良いワークスペースのやさしい雰囲気を表現しました。
施設案内には写真とアイコンを用いて、新規顧客へサービスの充実度をアピール。
店舗のデザインを総合的に請け負っているという設定で、メニュー表やオープンチラシは実際に店舗で使用している想定で作成しました。

制作範囲

企画 / デザイン / コーディング / ロゴ作成

制作期間

1ヶ月

使用ツール

Photoshop / Illustrator / Visual Studio Code /

使用言語

HTML / CSS / JavaScript
